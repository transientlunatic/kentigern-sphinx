======================
Step-by-step Tutorials
======================

``sphinx-kentigern`` comes with special support for the Sphinx ``steps`` directive, which is used to create
a step-by-step guide. The directive is used as follows:

.. steps:: A step-by-step guide

   .. step::  This is the first step.

        Description of the first step.

   .. step::  This is the second step.

        Description of the second step.

   .. step::  This is the third step.

        Description of the third step.
